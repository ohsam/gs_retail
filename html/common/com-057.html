이용안내
배송하신 물품의 배송정보는 배송상태에서 확인하시고 배송상태를 클릭하면 상세 배송정보를 확인 할 수 있습니다.
상세접수정보의 확인, 접수취소를 원하실 경우 로지아이 접수번호 부분을 클릭해주세요.
배송지연 문의는 택배사 고객센터로 연락해 주시기 바랍니다.
결제수단 별 접수취소 안내
택배접수취소는 취소접수 후 택배사 데이터 확인 후 영업일 기준 2일 이내 취소여부 확인 가능하며, 취소완료 이후 신용카드 결제 취소됩니다.
접수 취소 이후 자세한 결제 취소 문의는 고객행복센터 (1644–5425)으로 문의해주시기 바랍니다.
